# 실시간 여행 n빵

로그인 없이 하나의 링크로 같이 보는 여행 정산 웹앱입니다.

## 실행

```bash
./start-local.sh
```

브라우저에서 `http://localhost:4173`을 열면 새 여행 링크가 자동으로 만들어집니다.

`node`와 `npm`이 이미 설치되어 있다면 `npm start`로도 실행할 수 있습니다.

친구들이 같은 와이파이에 있다면 `localhost` 대신 이 컴퓨터의 내부 IP를 넣은 링크를 공유하면 됩니다. 외부 인터넷에서도 열려야 한다면 이 Node 앱을 Render, Railway, Fly.io 같은 서버에 올리면 됩니다.

## 배포

고정 링크로 오래 쓰려면 GitHub에 코드를 올린 뒤 Railway 같은 서버에 배포합니다.

초심자용 순서는 [DEPLOYMENT.md](./DEPLOYMENT.md)를 보면 됩니다.

배포 환경에서 데이터가 유지되게 하려면 `DATA_DIR` 환경변수를 Volume이 붙은 경로로 설정하세요.

## 기능

- 링크 기반 여행 정산방
- 친구 추가
- 지출별 결제자 선택
- 지출별 n빵 참여자 선택
- 실시간 갱신
- 누가 누구에게 얼마 보내면 되는지 송금표 계산
- 기본은 `data/trips.json`에 저장
- 배포 시 `DATA_DIR` 환경변수로 저장 위치 변경 가능
