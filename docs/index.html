Setup

Supabase 연결이 필요합니다

GitHub Pages에서 쓰려면 Supabase 프로젝트 URL과 anon key를 docs/config.js에 넣어야 합니다.

  1. Supabase에서 새 프로젝트를 만듭니다.
  2. supabase/schema.sql 내용을 SQL Editor에서 실행합니다.
  3. docs/config.js의 값을 프로젝트 URL과 anon key로 바꿉니다.
  4. 변경사항을 GitHub에 push하고 GitHub Pages를 켭니다.
